
# 1211. Queries Quality and Percentage

[View problem on the Leetcode](https://leetcode.com/problems/queries-quality-and-percentage/)

```sql
select query_name,
    round(
        avg(rating / position), 2) as 'quality',
    round(
        avg(rating < 3) * 100, 2) as 'poor_query_percentage'
from Queries
group by query_name;
```

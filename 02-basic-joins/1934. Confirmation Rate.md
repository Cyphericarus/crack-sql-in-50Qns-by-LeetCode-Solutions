# 1934. Confirmation Rate

[View problem on the Leetcode](https://leetcode.com/problems/confirmation-rate/)

```sql
SELECT 
    s.user_id, 
    ROUND(
      IFNULL(
        sum(c.action='confirmed')/count(action),
        0), 
      2) AS confirmation_rate
FROM signups s
LEFT JOIN confirmations c ON s.user_id = c.user_id
GROUP BY s.user_id;
```
